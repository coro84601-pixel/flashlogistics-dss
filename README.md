# FlashLogistics DSS

## Descripción del Proyecto

FlashLogistics enfrenta problemas de entregas tardías, incremento de costos de combustible y falta de información para clientes y gerentes. Para solucionar estos problemas se propone el desarrollo de un Sistema de Soporte a la Decisión (DSS).

## Objetivo SMART

Desarrollar un Sistema de Soporte a la Decisión (DSS) para FlashLogistics que reduzca las entregas tardías del 30% al 10%, disminuya los costos de combustible en un 15% y reduzca las consultas de clientes en un 70% mediante monitoreo GPS y optimización de rutas en un período de 3 meses.

## Canvas MVP

| Elemento            | Descripción                                                |
| ------------------- | ---------------------------------------------------------- |
| Problema            | Entregas tardías y falta de control                        |

| Usuarios            | Gerente de Operaciones, Despachador, Conductor y Cliente   |
| Funcionalidades MVP | GPS, rutas optimizadas, seguimiento de pedidos y dashboard |
| KPI 1               | Reducir retrasos del 30% al 10%                            |
| KPI 2               | Reducir costos de combustible en 15%                       |
| KPI 3               | Reducir llamadas de clientes en 70%                        |
| Tiempo              | 3 meses                                                    |
